# 空格

多个空格会被合并成一个，要想实现多个空格，使用以下占位符替代空格

`&nbsp;`输入一个英文空格
`&ensp;`输入一个中文空格
`&emsp;`输入两个中文空格

代码块里可以直接多个空格

# 换行

在一行的末尾添加两个或多个空格，然后按回车键,即可创建一个换行(`<br>`)。
为了兼容性，请在行尾添加“结尾空格”或 HTML 的 `<br>` 标签来实现换行。
两个或多个空格进行换行，称为 `结尾空格（trailing whitespace)` 的方式

代码块里可以直接换行

# 段落

连续两个或多个回车为另起一段

两个段落间要想空多行，每个空行用`<br>`占据

看了看github，大部分人喜欢用段落代替换行
